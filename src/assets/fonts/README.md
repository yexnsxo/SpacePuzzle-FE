# 폰트 파일

이 폴더에 다음 폰트 파일을 넣어주세요:

## 🔤 픽셀 폰트

### `pixel-font.ttf` 또는 `pixel-font.woff2`
- **설명**: 픽셀 아트 스타일 폰트
- **추천 폰트**:
  - Press Start 2P (Google Fonts에서 무료)
  - PixelMPlus
  - VT323
  - Silkscreen
- **용도**: 게임 제목, UI 텍스트
- **형식**: .ttf, .woff, .woff2 중 하나

## 📥 폰트 다운로드 방법

### 방법 1: Google Fonts 사용 (추천)
1. https://fonts.google.com/specimen/Press+Start+2P 접속
2. 다운로드 버튼 클릭
3. .ttf 파일을 이 폴더에 복사

### 방법 2: CDN 사용
- HTML에 직접 링크 추가 (파일 다운로드 불필요)
- 나중에 코드에서 알려드릴게요

## 💡 참고

- 한글 지원 폰트가 필요하면 "둥근모꼴" 같은 픽셀 한글 폰트 사용
- 또는 제목만 영문으로 하고 본문은 일반 폰트 사용
