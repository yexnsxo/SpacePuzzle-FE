# 천체 실루엣 이미지 폴더

이 폴더에 천체 선택 화면용 **흑백 실루엣** 이미지를 넣어주세요.

## 📸 파일명 규칙:

- `mercury-silhouette.png` (수성)
- `venus-silhouette.png` (금성)
- `earth-silhouette.png` (지구) ⭐ 우선 필요
- `mars-silhouette.png` (화성)
- `jupiter-silhouette.png` (목성)
- `saturn-silhouette.png` (토성)

## 🎨 이미지 특징:

- **스타일**: 흑백, 실루엣 (배경 투명)
- **형식**: PNG (투명 배경 필요)
- **크기**: 300x300px ~ 500x500px 권장
- **색상**: 검은색 또는 회색 톤

## 💡 용도:

천체 선택 화면(GamePlay)에서 아직 클리어하지 않은 천체를 표시할 때 사용됩니다.
클리어하면 컬러 이미지로 바뀝니다.

## 예시:

```
잠긴 상태: earth-silhouette.png (흑백)
해금 상태: earth.jpg (컬러)
```
