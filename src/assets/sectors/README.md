# 섹터 아이콘 이미지 폴더

이 폴더에 조종실에서 보이는 **섹터 대표 이미지**를 넣어주세요.

## 📸 파일명 규칙:

- `solar-system.png` (태양계) ⭐ 우선 필요
- `galaxy.png` (은하계)
- `nebula.png` (성운)
- `asteroid-belt.png` (소행성 지대)
- `black-hole.png` (블랙홀) - 특별 스테이지용

## 🎨 이미지 특징:

- **크기**: 아무거나 OK! (400x400px 이상 권장)
- **형식**: PNG 권장 (투명 배경 가능)
- **스타일**: 화려하고 눈에 띄는 이미지
- **특징**: 행성, 은하, 성운 등 각 섹터를 대표하는 이미지

## 💡 용도:

조종실 화면에서 클릭 가능한 큰 아이콘으로 표시됩니다.
예: 태양계 섹터를 클릭하면 → Sector 페이지로 이동

## 예시:

```javascript
import solarSystem from './sectors/solar-system.png';
```
