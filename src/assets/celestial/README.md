# 천체 퍼즐 이미지 폴더

이 폴더에 퍼즐용 천체 이미지를 넣어주세요.

## 📸 파일명 규칙:

### 태양계 행성:
- `mercury.jpg` (수성)
- `venus.jpg` (금성)
- `earth.jpg` (지구) ⭐ 우선 필요
- `mars.jpg` (화성)
- `jupiter.jpg` (목성)
- `saturn.jpg` (토성)
- `uranus.jpg` (우라누스)
- `neptune.jpg` (해왕성)
- `moon.jpg` (달)

### 기타 천체:
- `sun.jpg` (태양)
- `pluto.jpg` (명왕성)

## 🎨 이미지 권장 사양:

- **크기**: 아무거나 OK! (최소 500x500px 이상 권장)
- **형식**: JPG, PNG 모두 가능
- **비율**: 정사각형 권장 (1:1)
- **화질**: 고화질일수록 좋음

## ⚠️ 주의사항:

1. 파일명은 **반드시 소문자**로 작성
2. 공백 대신 **하이픈(-)** 사용
3. 확장자는 `.jpg` 또는 `.png`

## 🚀 사용 예시:

```javascript
import earth from './celestial/earth.jpg';
```

퍼즐 게임에서 이 이미지를 로드하여 퍼즐 조각으로 나눕니다.
