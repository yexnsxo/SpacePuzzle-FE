# 로그인 화면 이미지 에셋

이 폴더에 로그인 화면 관련 이미지 파일들을 넣어주세요!

## 🖼️ 추천 파일

### 1. `spaceship-interior-bg.png` 또는 `spaceship-interior-bg.jpg`
- **설명**: 우주선 내부 픽셀 아트 배경
- **용도**: 로그인 화면 전체 배경
- **권장 크기**: 1920x1080 이상
- **스타일**: 픽셀 아트, 우주선 내부 느낌

### 2. `login-panel.png` (선택사항)
- **설명**: 로그인 창 배경 패널
- **배경**: 투명 또는 반투명 PNG
- **용도**: Google 로그인 버튼 뒤 배경
- **스타일**: 픽셀 아트 패널/프레임

### 3. `control-panel.png` (선택사항)
- **설명**: 우주선 조종 패널
- **용도**: 장식 요소
- **스타일**: 픽셀 아트, 버튼/스위치/모니터
