# 랜딩 화면 이미지 에셋

이 폴더에 다음 이미지 파일들을 넣어주세요:

## 🚀 필수 이미지

### 1. `spaceship.png`
- **설명**: 메인 우주선 이미지 (픽셀 아트)
- **배경**: 투명 (PNG)
- **권장 크기**: 400-600px 너비
- **용도**: 화면 중앙에 배치, 위아래 흔들림 애니메이션

### 2. `play-button.png`
- **설명**: 플레이 버튼 (픽셀 스타일)
- **배경**: 투명 (PNG)
- **권장 크기**: 150-200px 너비
- **용도**: 우주선 아래 배치, 클릭 시 게임 시작

### 3. `play-button-hover.png` (선택사항)
- **설명**: 플레이 버튼 호버 상태
- **배경**: 투명 (PNG)
- **권장 크기**: play-button.png와 동일
- **용도**: 마우스 올렸을 때 표시
- **참고**: 없으면 CSS로 효과 처리 가능

## ✨ 선택사항 이미지

### 4. `title-logo.png` (선택사항)
- **설명**: 게임 제목 로고 (픽셀 폰트로 만든 이미지)
- **배경**: 투명 (PNG)
- **용도**: 우주선 위에 배치
- **참고**: 없으면 웹폰트로 텍스트 렌더링

### 5. `stars-layer.png` (선택사항)
- **설명**: 앞쪽 별 레이어 (패럴랙스 효과용)
- **배경**: 투명 (PNG)
- **크기**: 전체 화면 크기
- **용도**: 배경 위에 오버레이, 더 빠르게 움직여 depth 효과

### 6. `fallback-space.jpg`
- **설명**: NASA API 로딩 실패 시 대체 배경
- **형식**: JPG/PNG
- **크기**: 1920x1080 이상
- **용도**: 백업 우주 배경

---

## 📏 이미지 제작 가이드

- **픽셀 아트**: 8x8, 16x16 등 일정한 그리드 사용
- **색상 팔레트**: 통일된 컬러 사용 (우주 테마 - 파랑, 보라, 검정)
- **파일 형식**: PNG (투명 배경 필요한 것), JPG (배경)
- **최적화**: 파일 크기를 줄여서 로딩 속도 향상
