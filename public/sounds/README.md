# 사운드 파일

이 폴더에 다음 사운드 파일들을 넣어주세요:

## 🎵 BGM (배경음악)

### 1. `landing-bgm.mp3`
- **설명**: 랜딩 화면 배경음악
- **분위기**: 우주 테마, 차분하고 신비로운 느낌
- **길이**: 1-3분 (루프 재생)
- **볼륨**: 너무 크지 않게 (믹싱 시 -10dB 정도)
- **형식**: MP3 또는 OGG

## 🔊 효과음 (SFX)

### 2. `button-hover.mp3`
- **설명**: 버튼에 마우스 올렸을 때
- **길이**: 0.1-0.3초
- **느낌**: 짧고 가벼운 "띠링" 소리

### 3. `button-click.mp3`
- **설명**: 플레이 버튼 클릭 시
- **길이**: 0.3-0.5초
- **느낌**: 확실한 "딸깍" 또는 "펑" 소리

### 4. `zoom-transition.mp3`
- **설명**: 화면 확대되며 전환될 때
- **길이**: 1-2초
- **느낌**: 우주선 엔진 소리, "슈우우웅"

## 🎼 사운드 제작/다운로드 팁

### 무료 사운드 사이트
- **Freesound.org**: 효과음
- **Pixabay Music**: 배경음악
- **OpenGameArt.org**: 게임 음악/효과음
- **ZapSplat**: 다양한 효과음

### 검색 키워드
- BGM: "space ambient", "cosmic music", "sci-fi background"
- 효과음: "button click", "UI sound", "whoosh", "sci-fi transition"

## 📏 파일 규격

- **형식**: MP3 (호환성 최고) 또는 OGG
- **비트레이트**: 128kbps (효과음), 192kbps (BGM)
- **샘플레이트**: 44.1kHz
- **볼륨**: 정규화하여 일정하게 유지

---

## ⚠️ 저작권 주의

- 상업적 사용 가능한지 라이선스 확인
- 크레딧 표기가 필요한지 확인
- 프로젝트가 학교 과제면 보통 문제없음
